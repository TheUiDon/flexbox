Flexbox {Properties: Value;}

Flex-Direction
Flex Direction
"The flex-direction CSS property sets how flex items are placed in the flex container defining the main axis and the direction (normal or reversed)."
Flex-Wrap
Flex Wrap
"The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked."
Justify-Content
Justify Content
"The CSS justify-content property defines how the browser distributes space between and around content items along the x-axis of a flex container, and the inline axis of a grid container."
Align-Items
Align Items
"The CSS align-items property sets the align-self value on all direct children as a group. In Flexbox, it controls the alignment of items on the Y Axis."
Align-Content
Align Content
"The CSS align-content property sets the distribution of space between and around content items along a flexbox's cross-axis"
